# Password Generator

## Description

This project is a simple Password Generator created using Python and Tkinter as part of the CodSoft Python Internship.

The application generates a random password based on the length entered by the user.

## Features

* User-friendly GUI
* Generate random passwords
* User can choose the password length
* Uses letters, numbers, and special characters
* Displays the generated password instantly

## Technologies Used

* Python
* Tkinter
* Random module
* String module

## How to Run

1. Open the project folder.
2. Run the following command:

```bash
python password_generator.py
```
